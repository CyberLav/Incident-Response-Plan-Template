# Annex D — Communications and SITREP Template

## Standard SITREP (internal responders)

```
SITREP — [Incident ID] — [timestamp]
Status: [Active / Contained / Eradicating / Recovering / Closed]
Since last SITREP: [what changed]
Current actions: [what's happening now]
Blockers: [anything preventing progress]
Next SITREP due: [time]
```

## Executive Briefing

```
EXEC BRIEF — [Incident ID] — [timestamp]
Business impact: [plain-language, no jargon]
Current status: [ ]
Decisions needed from leadership: [ ]
Estimated time to resolution: [ ]
```
